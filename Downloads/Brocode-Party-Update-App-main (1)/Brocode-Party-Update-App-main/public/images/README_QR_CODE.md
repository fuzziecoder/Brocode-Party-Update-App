# Payment QR Code Image

Please add your payment QR code image to this directory:

**File name:** `payment-qr.png`

**Location:** `public/images/payment-qr.png`

**Requirements:**
- PNG format
- Square image (recommended: 500x500px or higher)
- White background
- QR code should be clearly visible
- UPI ID: godw1921-1@okicici

If the image is not found, the app will show a placeholder. Make sure to add the actual QR code image for the best user experience.
